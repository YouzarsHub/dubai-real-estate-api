# 🏙️ Dubai Luxury Real Estate Dashboard

A full-stack interactive web application built with Python and Streamlit designed to aggregate, filter, and manage luxury property listings in the Dubai real estate market. 

## 🔗 Live Application
👉 **[View Live App on Streamlit Cloud](https://dubai-real-estate-api-dzcuxkg5bgfpyagrluidx7.streamlit.app/)**

## 🚀 Key Features
* **Interactive Market Filters:** Dynamic filtering of premium listings by location (e.g., Dubai Marina, Palm Jumeirah), price thresholds, and property types.
* **Internal Client CRM System:** A stateful database simulation allowing real estate agents to log clients, track specific property viewings, and manage budget preferences.
* **Data-Driven UI:** Smooth integration of Pandas dataframes with high-performance UI components.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Streamlit (UI & Stateful App Architecture)
* **Data Manipulation:** Pandas
* **Deployment:** Streamlit Community Cloud
